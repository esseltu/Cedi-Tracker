# Cedi Tracker

A lightweight Progressive Web App (PWA) for tracking personal spending in Ghana Cedis (GHS).

## Features
- **Transaction Tracking**: Log income and expenses with categories.
- **Dashboard**: View current balance and credit score estimates.
- **Credit Score**: Visual gauge based on spending habits.
- **Offline Support**: Fully functional PWA.
- **Secure**: Powered by Firebase Auth and Firestore.

## Tech Stack
- React
- Tailwind CSS
- Firebase (Auth, Firestore)
- Vite
